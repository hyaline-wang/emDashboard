# emNaviBoard

emNaviBoard 是 emNaviDevSuite 的一部分，是基于vue.js构建的前端管理面板。

## 快速构建

首先安装 pnpm 和 node.js
```bash
git clone https://github.com/hyaline-wang/emNaviboard.git
cd emNaviboard
pnpm install 
# Compile and Hot-Reload for Development
pnpm dev
# Type-Check, Compile and Minify for Production
pnpm build
```
